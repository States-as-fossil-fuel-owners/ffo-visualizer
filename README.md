# Geoflow Visualizer

Visualizes any type of flow between countries.

## Requirements

Nodejs + npm

## Installation

`npm install`

## Run

`npm start`

## Usage

Use the file selector to select a CSV file that at least has the columns `source`, `target` and `weight`.
